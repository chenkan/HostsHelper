## HostsHelper - Android hosts小助手

Hosts-Helper是一个方便、简单的android手机hosts管理工具。

## Functions
* 罗列android安装机hosts列表；
* 新增host记录；
* 编辑、注释、删除单条host记录；
* 批量注释，删除多条host记录。
